The Movie Site - Inspired from a still image I seen of someone's front end design portfolio, I took that a step further and made a fully interactive site

I decided to step up the complexity for this project. The original inspiration allowed you to search the The Movie Database (TMDb) for details on a single movie. I took that a bit further and load the 'Now Playing' movies by default, displaying the first 16 listed. You can also switch between 'Popular', 'Upcoming', 'Now Playing' and 'Top Rated'.

You can also search, which supports auto complete, for a particular movie if you know it's name. Or clicking on any movie card from the earlier lists will being up more detailed information, along with reviews, the trailer, and if available, the cast information for the movie.

My first major project using a 3rd party API
A simple NodeJS project using EJS template engine that renders today's current weather with more details. Then a 7 day forecast with popup cards, when you hover on a card, it pushes it's neighbors away and raises the one you're hovering over up for easier visibility. It also sports a light and dark theme and a toggle to show how alerts would look. It also uses Skycons for animated weather icons. All the local weather data is aquired using the DarkSky. API
